Title:  Export to Web Presentation

Level:  2 - slide

Seq:    13

Class:  slide

Body:

+ Select *File -> Export*
+ Select *Web Presentation* as output format
+ Select output folder
+ Each  note/slide becomes a web page
+ Generated `index.html` file duplicates title slide
+ Double-click on the index file to view in your web browser
+ Can be shared on the web
	- But can also be viewed locally on your computer
