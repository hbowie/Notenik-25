Body:

The Image Name field can be used to identify an image to be incorporated into the Display. 

The Collection Settings must have *Enable Local Image References* checked. 

The Image Name Dark field can be used to identify an alternate image to be displayed with a Dark Appearance. 

(Or it can be used to specify the same images used for a non-Dark display.)

The Image Layout field can be used to control placement of an image within the display. 

See the next two slides for Image Layout examples.
