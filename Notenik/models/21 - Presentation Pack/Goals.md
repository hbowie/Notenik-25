Body:

Notenik can be used to craft and deliver a slide show to be presented to a group. 

This would be as an alternative to something like PowerPoint or Google Slides. 

By the time we're done, you'll see how we can use Notenik to:

+ Create and arrange a series of slides
+ Present those slides from the Notenik app itself
+ Export to a series of HTML files that can be presented using any web browser (Chrome, Safari, etc.)

*Caveat: Notenik is best used for simple, straightforward presentations that are primarily text-based.*
