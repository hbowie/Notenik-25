Title:  Why Markdown + Fields?

Level:  2 - slide

Seq:    5

Class:  slide

Body:

#### Markdown

+ Uses an easy-to-read, easy-to-write plain text format
+ Allows authors to focus on content first, formatting later
+ Neutral format that can be written and displayed using a variety of tools

#### Fields

+ Body field contains Markdown text to be displayed on each slide
+ Seq field keeps everything in desired sequence
+ Script field allows entry of speaker's notes
+ Class field provides for different types of content
+ Level field: 
	- provides hierarchy
	- determines heading weight
